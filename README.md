# Монеты → Гексаграмма

Мини‑приложение: 6 раз подкидываем по 3 монеты и рисуем гексаграмму.

- больше орлов (орлы/heads) среди 3 монет → цельная черта (1 “черта”)
- больше решек (решки/tails) среди 3 монет → разорванная черта (2 “черты”)

## Как запустить
Откройте файл `index.html` в браузере (например, двойным кликом).
